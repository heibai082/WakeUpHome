# 醒醒宅家 (WakeUpHome) v0.0.735 杰作锚定版

这是一套具备全权主宰感、工业级稳定性的家庭物资管理系统。

### 🌟 核心卖点
1. **主宰权 - 定义属于您的系统**：支持自定义系统全称，顶栏、标题、推送名同步修改。
2. **视觉权 - 全天候舒适度**：支持白天模式、黑夜模式及跟随系统外观切换。
3. **决策权 - 精准预警**：批次折叠聚合展示，支持单品独立设置提前 N 天预警，且支持全量二次编辑改名。
4. **生存权 - 数据永恒**：具备扫码自学习大脑、变动轨迹双轨日志、级联物理删除逻辑。

### 🚀 快速启动
1. 安装依赖：`pip install -r requirements.txt`
2. 运行后端：`uvicorn main:app --host 0.0.740 --port 8000`
3. 配置 Webhook：在设置中心配置 Lucky 推送地址。
